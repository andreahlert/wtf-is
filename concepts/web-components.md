# WTF are Web Components?

A set of browser-native APIs that let you create reusable custom HTML elements with encapsulated styles and behavior. They work in any framework or with no framework at all. The three main pieces are Custom Elements, Shadow DOM, and HTML Templates.

```
┌─────────────────────────────────┐
│        Web Components           │
│                                 │
│  Custom Elements  (define tags) │
│  Shadow DOM       (isolation)   │
│  HTML Templates   (reuse)       │
│                                 │
│  <my-button>Click</my-button>   │
└─────────────────────────────────┘
```

**Examples:** [Lit](https://github.com/lit/lit), [Stencil](https://github.com/ionic-team/stencil), [FAST](https://github.com/microsoft/fast)
