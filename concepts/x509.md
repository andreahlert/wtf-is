# WTF is X.509 Certificate?

An X.509 Certificate is a standardized digital document that binds a public key to an identity (like a domain name or organization). It's the format used by TLS/HTTPS certificates. It contains the subject (who it's for), the issuer (who signed it), validity dates, the public key, and a digital signature from a Certificate Authority.

```
┌─────────────────────────────────┐
│  X.509 Certificate              │
│                                 │
│  Subject:  CN=example.com       │
│  Issuer:   CN=Let's Encrypt     │
│  Valid:    2024-01-01 to         │
│            2025-01-01            │
│  Public Key: RSA 2048-bit       │
│  Signature: (CA's signature)    │
└─────────────────────────────────┘

  Common file extensions:
    .pem  .crt  .cer  .der  .p12
```

**Examples:** [Let's Encrypt](https://github.com/letsencrypt), [mkcert](https://github.com/FiloSottile/mkcert), [OpenSSL](https://github.com/openssl/openssl)
