# WTF is Deep Learning?

Deep learning is machine learning using neural networks with many layers (hence "deep"). More layers let the network learn increasingly abstract representations, from edges to shapes to objects. It's what powers image recognition, language models, and most modern AI.

```
Shallow (2 layers)          Deep (many layers)

Input → [H] → Output       Input → [H1] → [H2] → [H3] → ... → [Hn] → Output
                                     │       │       │              │
                                   edges   shapes  parts         objects
```

**Examples:** [PyTorch](https://github.com/pytorch/pytorch), [Keras](https://github.com/keras-team/keras), [FastAI](https://github.com/fastai/fastai)
