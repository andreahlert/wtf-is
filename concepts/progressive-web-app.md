# WTF is Progressive Web App (PWA)?

A website that can behave like a native app. It can be installed on your home screen, work offline, send push notifications, and load fast on bad connections. Under the hood, it uses Service Workers, a Web Manifest, and HTTPS. It is still a website, just with superpowers.

```
Regular Website:              PWA:

Browser tab only              Installable on home screen
Online only                   Works offline
No push notifications         Push notifications
URL bar visible               Full-screen mode
```

**Examples:** [Twitter/X](https://x.com), [Starbucks](https://app.starbucks.com), [Workbox](https://github.com/GoogleChrome/workbox)
