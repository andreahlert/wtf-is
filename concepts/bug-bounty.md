# WTF is Bug Bounty?

A Bug Bounty program pays external security researchers for finding and responsibly reporting vulnerabilities in your product. Instead of hoping nobody finds your bugs, you incentivize skilled hackers to find them first and tell you. Payouts range from $50 for low-severity to $100k+ for critical findings.

```
┌────────────┐   finds bug    ┌──────────────┐
│ Researcher │──────────────▶ │ Bug Bounty   │
│ (hacker)   │                │ Platform     │
└────────────┘                └──────────────┘
                                    │
                              validates + triages
                                    │
                                    ▼
                              ┌──────────────┐
                              │   Company    │
                              │ fixes bug,   │
                              │ pays bounty  │
                              └──────────────┘
```

**Examples:** [HackerOne](https://www.hackerone.com/), [Bugcrowd](https://www.bugcrowd.com/), [Huntr](https://github.com/418sec/huntr)
