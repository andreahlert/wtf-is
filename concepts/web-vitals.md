# WTF are Web Vitals?

A set of metrics defined by Google that measure real-world user experience on a web page. They quantify how fast content appears, how quickly the page responds to interaction, and how stable the layout is. Google uses them as ranking signals for search results.

```
Core Web Vitals:

LCP  (Largest Contentful Paint)  ──► Loading speed
INP  (Interaction to Next Paint) ──► Responsiveness
CLS  (Cumulative Layout Shift)   ──► Visual stability

Good:   LCP < 2.5s, INP < 200ms, CLS < 0.1
```

**Examples:** [web-vitals](https://github.com/GoogleChrome/web-vitals), [Lighthouse](https://github.com/GoogleChrome/lighthouse), [PageSpeed Insights](https://pagespeed.web.dev)
