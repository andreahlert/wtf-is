# WTF is an IDE?

Integrated Development Environment. A text editor with built-in tools for coding: syntax highlighting, debugging, autocompletion, version control, terminal, and more. All in one window instead of separate tools.

```
+------------------------------------------+
| IDE                                      |
| +--------+ +------------------+ +------+ |
| | File   | | Editor           | | Term | |
| | Tree   | |                  | | inal | |
| |        | | code + highlight | |      | |
| |        | | autocomplete     | | $    | |
| |        | | inline errors    | |      | |
| +--------+ +------------------+ +------+ |
| [ Debug ] [ Git ] [ Extensions ] [ Run ] |
+------------------------------------------+
```

**Examples:** [VS Code](https://github.com/microsoft/vscode), [IntelliJ IDEA](https://www.jetbrains.com/idea/), [Zed](https://github.com/zed-industries/zed)
