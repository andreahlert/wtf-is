# WTF is Design System?

A collection of reusable components, design tokens, guidelines, and documentation that ensures a consistent look and behavior across an entire product or organization. It is the single source of truth for how things should look and work, shared between designers and developers.

```
Design System
├── Tokens (colors, spacing, fonts)
├── Components (Button, Input, Modal)
├── Patterns (forms, navigation, layouts)
├── Guidelines (accessibility, voice, motion)
└── Documentation (how and when to use each piece)
```

**Examples:** [Material UI](https://github.com/mui/material-ui), [Radix](https://github.com/radix-ui/primitives), [Chakra UI](https://github.com/chakra-ui/chakra-ui)
