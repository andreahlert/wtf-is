# WTF is NLP?

NLP (Natural Language Processing) is the field of AI that deals with understanding, interpreting, and generating human language. It covers everything from spell checking to machine translation to chatbots. Before transformers, NLP relied heavily on hand-crafted rules and statistical methods; now it's mostly deep learning.

```
  NLP tasks:
  ┌────────────────────────────────────────────┐
  │ Text in ──► ┌──────────────────────┐       │
  │             │ Tokenize             │       │
  │             │ Classify / Translate │       │
  │             │ Summarize / Generate │       │
  │             │ Extract entities     │       │
  │             │ Answer questions     │       │
  │             └──────────────────────┘ ──► out│
  └────────────────────────────────────────────┘
```

**Examples:** [spaCy](https://github.com/explosion/spaCy), [NLTK](https://github.com/nltk/nltk), [Hugging Face Transformers](https://github.com/huggingface/transformers)
