# WTF is AI Safety?

AI safety is the field focused on making AI systems behave as intended and not cause harm. It covers preventing models from generating dangerous content, ensuring they follow instructions faithfully, making them robust against adversarial attacks, and thinking about long-term risks from increasingly capable systems.

```
  AI Safety concerns:
  ┌────────────────────────────────────────┐
  │ Near-term                              │
  │ ├─ Harmful content generation          │
  │ ├─ Bias and discrimination             │
  │ ├─ Privacy violations                  │
  │ └─ Prompt injection attacks            │
  │                                        │
  │ Long-term                              │
  │ ├─ Loss of human control               │
  │ ├─ Misaligned goals                    │
  │ └─ Autonomous self-improvement         │
  └────────────────────────────────────────┘
```

**Examples:** [Anthropic Safety](https://www.anthropic.com/research), [OpenAI Safety](https://openai.com/safety), [LMSYS Chatbot Arena](https://github.com/lm-sys/FastChat)
