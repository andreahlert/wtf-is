# WTF is Localization (l10n)?

The process of actually translating and adapting your app for a specific language or region. While internationalization (i18n) sets up the infrastructure, localization fills in the actual translations, date formats, currency symbols, and cultural conventions. The "10" in "l10n" is the number of letters between "l" and "n."

```
i18n (infrastructure):        l10n (actual work):

Set up translation files      Translate strings to Portuguese
Use {t('key')} in code        Format dates as DD/MM/YYYY
Support RTL layouts           Use R$ for currency
Handle pluralization rules    Adapt images with text
```

**Examples:** [Crowdin](https://crowdin.com), [Lokalise](https://lokalise.com), [Transifex](https://www.transifex.com)
