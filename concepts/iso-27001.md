# WTF is ISO 27001?

ISO 27001 is an international standard for information security management systems (ISMS). It provides a framework for systematically managing sensitive information through risk assessments, security controls, and continuous improvement. Getting certified means an accredited auditor verified your security program meets the standard. Many enterprise contracts require it.

```
ISO 27001 structure:

  ┌──────────────────────────────┐
  │  ISMS (your security system) │
  │                              │
  │  1. Risk assessment          │
  │     → identify threats       │
  │     → evaluate likelihood    │
  │                              │
  │  2. Controls (Annex A)       │
  │     → 93 controls across     │
  │        4 categories          │
  │                              │
  │  3. Continuous improvement   │
  │     → internal audits        │
  │     → management review      │
  │     → corrective actions     │
  └──────────────────────────────┘

  Certification: 2-stage audit by accredited body
```

**Examples:** [Vanta](https://www.vanta.com/), [Drata](https://drata.com/), [ISO 27001 standard](https://www.iso.org/standard/27001)
