## Format

Every explanation follows the same rules:

1. **Max 3-4 sentences** to explain the concept
2. **ASCII diagram** when it helps (optional)
3. **Real-world examples** of tools/products that implement it
4. **No prerequisites**. If you need to understand another concept first, the explanation is too complicated.
